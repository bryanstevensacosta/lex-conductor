✅ Not a resident of U.S. embargoed countries
✅ Not restricted by local laws from participating

❌ **INELIGIBLE** if you are:
- Employee of IBM Group (including Red Hat)
- Government employee (any level)
- Employee of advertising/promotion agencies
- Employee of contest organizers (including BeMyApp)
- Immediate family member of above (parents, children, siblings, spouse)
- Living in same household as above

✅ **ELIGIBLE** if you are:
- University/college student (even if employed part-time by institution)
- Independent developer
- Startup employee
- Freelancer

## Security & Credentials

### API Key Management
⚠️ **CRITICAL**: Never expose IBM Cloud credentials in public repositories

**If credentials are detected publicly:**
1. Credential immediately deactivated
2. Account suspended
3. Must remove from ALL public sources
4. Must rotate credentials
5. Contact support to restore access
6. May affect contest eligibility

### Best Practices
✅ Use environment variables
✅ Use .env files (never commit)
✅ Configure .gitignore properly
✅ Use .env.example for templates
✅ Rotate keys regularly
✅ Use separate keys for dev/prod

### What NOT to Commit
❌ .env files
❌ API keys
❌ Access tokens
❌ Passwords
❌ Private keys
❌ Credentials of any kind
