# IBM Dev Day: AI Demystified Hackathon

## Event Overview

**IBM Dev Day: AI Demystified** is a free virtual hackathon event designed for developers, engineers, and AI builders.

- **Date**: January 29, 2026 (Event) | Work Period: Jan 30 - Feb 1, 2026
- **Submission Deadline**: February 1, 2026 - 10:00 AM ET (HARD DEADLINE)
- **Format**: Virtual event
- **Platform**: BeMyApp
- **Theme**: "AI Demystified — From idea to deployment"

## Event Focus

The hackathon programming is designed to help participants navigate the complex AI landscape through:

- Developer tools and infrastructure automation
- AI agent building and governance best practices
- Multi-agent workflows
- IBM watsonx and Granite model integration
- Open-source AI tools and projects

## CRITICAL REQUIREMENTS (MANDATORY)

### 1. IBM watsonx Orchestrate (REQUIRED)
⚠️ **ALL SUBMISSIONS MUST USE IBM WATSONX ORCHESTRATE TO BE ELIGIBLE FOR PRIZES**

Your solution MUST:
- Use watsonx Orchestrate as the core orchestration platform
- Build agents using the Agent Development Kit (ADK) or no-code builder
- Demonstrate agent collaboration through Orchestrate
- Show clear integration in video demo and documentation

### 2. Agentic AI Implementation
- Multiple AI agents working together
- Clear agent roles and responsibilities
- Agent-to-agent communication
- Autonomous decision-making capabilities

### 3. Submission Deliverables (ALL REQUIRED)

**By February 1, 2026 - 10:00 AM ET:**

1. **Video Demo** (Max 3 minutes)
   - Problem explanation
   - Live demo showing watsonx Orchestrate (≥90 seconds of screen demo)
   - Solution walkthrough
   - Narration throughout
   - Public YouTube/Vimeo link

2. **Problem & Solution Statement** (≤500 words)
   - Problem description
   - Solution overview
   - Target users
   - Innovation/creativity
   - Real-world impact

3. **Agentic AI + watsonx Orchestrate Statement**
   - How watsonx Orchestrate is used (CRITICAL)
   - List of all agents
   - What each agent does
   - How agents collaborate
   - Integration details

4. **Code Repository** (Optional but recommended)
   - Public GitHub/GitLab/Bitbucket link
   - Working proof-of-concept
   - Setup instructions
